# Personal Notes App

Website ini merupakan submission kursus  [Belajar Fundamental Aplikasi Web dengan React](https://www.dicoding.com/academies/413). Di website ini terdapat fitur login dan register, dark mode dan light mode, translate bahasa Inggris dan Indonesia. Website ini dapat menambahkan catatan, melihat catatan, mencari catatan, mengarsipkan catatan, dan menghapusnya.
